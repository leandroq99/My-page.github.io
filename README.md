# Estudo de Tecnologia

Este repositório é usado como espaço de estudo e registro pessoal sobre várias tecnologias e ferramentas de infraestrutura.

## Objetivo

Documentar e organizar conceitos, exemplos e experimentos em tecnologias como:

- Docker
- Ansible
- Automação de flashcards para Anki
- Estudos e anotações gerais

## Estrutura do repositório

- `Docker/`
  - `Docker_notes.txt` — notas sobre comandos e conceitos básicos de Docker
- `Ansible/`
  - Playbooks de exemplo para operações com AWS EC2, AMI, Auto Scaling, snapshots e validação de checkpoints
- `Anki/`
  - `anki_gen.py` — gerador de flashcards para Anki
  - `README_anki_gen.md` — instruções de uso do gerador e exemplos de importação
  - Documentos e arquivos de apoio para estudo
- `main1`, `main2`, `teste`, `file`
  - Arquivos e rascunhos diversos usados durante os estudos

## O que estudar aqui

- Conceitos de containerização e comandos práticos do Docker
- Automação e orquestração com Ansible
- Criação e importação de flashcards para Anki
- Experimentos e anotações em arquivos soltos

## Como usar

1. Abra este repositório no VS Code ou outro editor de texto.
2. Navegue pelas pastas `Docker`, `Ansible` e `Anki`.
3. Leia os arquivos de notas e documentação para consolidar os conceitos.
4. Execute scripts e teste os exemplos quando quiser praticar.

## Observações

- Este repositório é voltado para estudos pessoais e aprendizado contínuo.
- Os arquivos podem ser atualizados conforme novos tópicos e experimentos forem adicionados.
- Use esta base para acompanhar seu progresso em tecnologia.
